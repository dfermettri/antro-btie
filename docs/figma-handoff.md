## Current access

Public oEmbed is available and provides a thumbnail. The Figma API endpoint for
node data returns `403 Invalid token`, so this project cannot yet read exact
frames, layer names, typography styles, coordinates, or export original assets.
The downloaded `.fig` archive contains `canvas.fig` in binary `fig-kiwi` format,
so it is useful for embedded PNG assets but not enough for reliable layer
inspection from the terminal.
